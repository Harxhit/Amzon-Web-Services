# Backend Overview

The backend of Crudder is designed as a scalable, API-driven service supporting authentication, posting, blogging, notifications, trending logic, and user features. It is built entirely with TypeScript to ensure type safety and maintainability.

## Technologies Used

### Core Backend

* **Node.js**: Runtime environment for executing server-side JavaScript.
* **Express**: Web framework used to define routes, middleware, and API structure.
* **TypeScript**: Strongly typed language on top of JavaScript for cleaner and safer backend code.

### AWS (Used for Learning and Deployment)

## Backend Capabilities

* User authentication (sign up, sign in)
* CRUD operations for posts
* Blog creation and retrieval
* Trending feed and home feed logic
* Notifications system
* User profiles and related data handling

## Architectural Highlights

* RESTful API structure
* Modular routing and controller organization
* Middleware for validation and authentication
* Clear separation of concerns between routes, controllers, and data logic
* TypeScript-based interfaces and types for strong compile-time safety
